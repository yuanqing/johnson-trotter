> An implementation of the [Johnson-Trotter](http://en.wikipedia.org/wiki/Steinhaus-Johnson-Trotter_algorithm) permutation algorithm.

## Installation

Install via [npm](https://npmjs.com/):

```
$ npm i --save johnson-trotter
```
